.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	struct :ref:`discord_message<doxid-structdiscord__message>`* disco_channel_send_message(
		:ref:`bot_client_t<doxid-disco_8h_1ab503de679841020121923d5f2daaca09>`* bot,
		char* content,
		uint64_t channel_id,
		struct :ref:`discord_create_message<doxid-structdiscord__create__message>`* message,
		bool return_struct
	)

Sends a message and if enabled also returns the sent message struct.



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- bot

		- Active bot instance

	*
		- content

		- Message content to send (if empty, embed or components need to exist)

	*
		- channel_id

		- ID of the channel the message should be sent in

	*
		- message

		- A :ref:`discord_create_message <doxid-structdiscord__create__message>` struct. Can be NULL

	*
		- return_struct

		- Bool if the sent message should be returned or not (if yes, it has to be destroyed with :ref:`disco_destroy_message() <doxid-group___message_1gaf8c130cb715240bb658ac80f52d3fed2>` after)



.. rubric:: Returns:

struct discord_message\* NULL or the struct of the sent message

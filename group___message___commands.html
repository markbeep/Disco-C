Fetches the message from Discord without first checking if it's in the cache.

The message will be added to the cache if it doesn't exist yet. discord_get_message is preferred over this fetch function, because this will force a new entry to the cache on every call. Deleted messages won't be retrieved.

Parameters
botActive bot instance.
channel_idChannel to fetch message from.
message_idMessage to fetch.

Returns an array of Discord message structs.

The parameters around, before and after are mutually exclusive, only one may be passed at a time. All the retrieved messages will be placed into the cache. NOTE: Messages don't have to be destroyed as they are all in the cache. Only the array itself has to be freed.

Parameters
botActive bot instance.
channel_idChannel to fetch messages from.
limitMax number of messages to return (1-100). 0 to ignore and use the default of 50.
aroundGet messages around this message ID. 0 to ignore.
beforeGet messages before this message ID. 0 to ignore.
afterGet messages after this message ID. 0 to ignore.
cbCallback struct which is called once ther request is done.

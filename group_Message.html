struct discord_message* discord_channel_send_message(
    bot_client_t* bot,
    char* content,
    uint64_t channel_id,
    struct discord_create_message* message,
    bool return_struct
)

Sends a message and if enabled also returns the sent message struct.

Parameters:

bot

Active bot instance

content

Message content to send (if empty, embed or components need to exist)

channel_id

ID of the channel the message should be sent in

message

A discord_create_message struct. Can be NULL

return_struct

Bool if the sent message should be returned or not (if yes, it has to be destroyed with discord_destroy_message() after)

Returns:

struct discord_message* NULL or the struct of the sent message
